How to join

The official home of The Bob SMP

Java Edition

Open Multiplayer in Minecraft Java, click Add Server, and use:

Address: java.thebobsmp.com:40289

  • Version: use the server’s listed version (or latest compatible).
  • Make sure you have a stable internet connection.
  • Read the rules before you join.

Bedrock Edition

On Bedrock (Windows 10, mobile, console), add a new external server:

Address: play.thebobsmp.com
Port: 2327

  • Make sure “Use External Server” is supported on your platform.
  • Check that you typed the address and port exactly.

Discord & whitelisting

Some SMPs require whitelisting or application before joining. If The Bob SMP uses this, you can handle it through the Discord server.
