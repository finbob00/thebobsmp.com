The staff team keeps the server fair, stable, and enjoyable for everyone. If you need help, have an issue, or want to report something, these are the people to contact.

Contacting staff

  • Use the in-game report or help command if available.
  • Open a ticket in the Discord server for serious issues.
  • Do not DM staff repeatedly for minor issues; use the proper channels.
